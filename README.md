# Tablet Consoles Project

Цей проєкт показує порівняння різних планшетних консолей.  
Він створений за допомогою HTML, CSS та JavaScript і включає таблиці та інтерактивні елементи для наочного відображення даних.

## Структура проєкту

- `index.html` — головна сторінка проєкту  
- `style.css` — стилі для сторінки  
- `script.js` — JavaScript для взаємодії та динаміки  

## Як переглянути

1. Відкрий файл `index.html` у будь-якому браузері.  
2. Переглядай таблиці та дані, взаємодій з елементами сторінки.

## Мета проєкту

- Порівняти характеристики різних планшетних консолей  
- Покращити навички роботи з HTML, CSS та JavaScript  
- Створити наочну та зручну для користувача демонстрацію
